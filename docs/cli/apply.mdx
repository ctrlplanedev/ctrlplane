The `ctrlc apply` command creates or updates resources in Ctrlplane from YAML
definition files.

## Usage

```bash
ctrlc apply -f <file>
```

## Flags

| Flag     | Short | Required | Description                               |
| -------- | ----- | -------- | ----------------------------------------- |
| `--file` | `-f`  | Yes      | Path to the YAML resource definition file |

## Example

```bash
ctrlc apply -f resource.yaml
```
